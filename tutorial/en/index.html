NoneOS

Welcome to NoneOS - a browser-based virtual operating system. You can create independent virtual system instances on multiple devices and achieve seamless connections between these systems through peer-to-peer technology. All data interactions occur directly between devices without going through any centralized server.

NoneOS Features Overview

The system comes with multiple built-in utilities. You can start experiencing with the following features:

Personalized Navigation Homepage

  1. Click the circular icon on the left or bottom to expand the application list
  2. Right-click on web favorites and select "Open in new tab"
  3. Set this page as your browser homepage

Between successfully paired virtual systems, you can click the sync button in the upper left corner to select other devices and view their favorite content.

Cross-Device File Transfer

In the "File Manager" application, you can directly transfer files between paired devices. Supports importing files from the real system to the virtual system before transferring.

Device Pairing Guide

Establish peer-to-peer data connections through device pairing:

  1. Ensure both devices have NoneOS installed
  2. On both devices:
    • Open the application list
    • Launch the "LinkMe" application
    • Click the "Add Device" button
  3. Exchange and enter pairing codes
  4. Both parties confirm pairing
  5. Device identifiers will be displayed after successful pairing
